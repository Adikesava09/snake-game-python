# 🐍 Realistic Snake Game (Python)

A realistic snake game built using Python Turtle Graphics.

## 🚀 Features
- Elongated natural snake body
- Score & High Score system
- Wall collision detection
- Self collision detection
- Game Over screen

## 🛠 Technologies Used
- Python
- Turtle Graphics

## ▶ How to Run
1. Install Python
2. Run: python snake_game.py
